# 🧠 AI Text Summarizer System

A Flask-based web application that performs **text summarization using NLP and Scikit-Learn** and calculates similarity accuracy using Fuzzy Logic.

---

## 🚀 Features
- Accept long text input
- Generate short meaningful summary
- Display word count (Original vs Summary)
- Fuzzy logic based accuracy score
- Simple and clean dashboard UI

---

## 🛠️ Tech Stack
- Flask
- Python (NLTK, Scikit-learn)
- FuzzyWuzzy (Similarity Score)
- HTML, CSS
